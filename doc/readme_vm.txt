========================================
NanoDefiner e-tool VM setup instructions
========================================

- import appliance into VirtualBox and start the VM
- login using username nanodefiner and password nanodefiner
- if necessary, change the keyboard layout using the "keyboard-layout.sh"
	script on the desktop (default is QUERTZ without dead keys)
- change passwords: it is recommended to change the following passwords
	- nanodefiner user (default password: nanodefiner), change using the desktop shortcut "change-pw.sh"
	- MySQL root user (no password by default)
	- MySQL nanodefiner user (default password: nanodefiner)
- see UPGRADE.txt for upgrade instructions
